Next.js 是一个用于构建全栈 Web 应用程序的 React 框架。它提供了许多开箱即用的功能，如服务器端渲染 (SSR)、静态站点生成 (SSG) 和 API 路由。

### 为什么选择 Next.js？

*   **性能优化：** 自动代码分割和图像优化。
*   **SEO 友好：** 服务器端渲染让搜索引擎更容易抓取内容。
*   **开发体验：** 快速刷新 (Fast Refresh) 和强大的路由系统。
